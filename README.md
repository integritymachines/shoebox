# Shoebox

**A bare-bones governance structure based on organizational primitives.**

MIT licensed • Single-file protocol • Designed for human + LLM agent companies.

## What is the shoebox?

The shoebox is a minimal, integrity-focused governance framework and operating protocol for AI-augmented teams. Think of it as the tiniest possible ERP.

It defines top-level roles:

- **A** = Architect (you — the human owner of the fixed `A_kernel`)
- **H** = Helmsman (the single operational interface with A)
- **B/S/R** = Builder / Seller / Runner (the execution agents)

Everything runs through strict **Plan → Do → Check** cycles, versioned artifacts, six verdict fields (V1–V6), and crystal-clear decision gates (GO/NO/RE/FW). Zero drift is enforced by design.
